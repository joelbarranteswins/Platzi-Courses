### Proceso del machine learning

El proceso de machine learning generalmente se divide en tres etapas: entrenamiento, validaci√≥n y prueba.

### Entrenamiento

En la etapa de entrenamiento, el algoritmo de machine learning recibe un conjunto de datos de entrenamiento que incluye caracter√≠sticas y etiquetas. ### Validaci√≥n

Si la precisi√≥n del modelo no es satisfactoria, se ajustan los par√°metros del algoritmo y se repite la etapa de entrenamiento.

### Prueba
